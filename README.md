# pfe_ft_nmap

School project scaffold inspired by ft_nmap.

## Structure
- include/: public headers (.hpp)
- src/: implementation (.cpp)
  - parser/, send/, receive/, utils/

## Build
For CLI version, run :
```sh
make
```
For GUI version, run :
```sh
make gui
```
To run CLI version : 
```sh
./pfe_ft_nmap
```
To run GUI version : 
```sh
./pfe_ft_nmap_gui
```
